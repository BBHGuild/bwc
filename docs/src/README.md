# Documentation

Welcome to the documentation for the Bounty Hunting skill. This knowledge base provides essential resources, classifications, and logs to assist in finding vulnerabilities in EVM smart contracts and responding to security incidents.

## Contents

- **Bounty Hunting Skill**: Comprehensive instructions and workflows for bug hunting and incident response.
- **References**:
    - **Blockchain Weakness Classification (BWC)**: A taxonomy for classifying smart contract vulnerabilities.
    - **Immunefi Severity Classification (IVSCS)**: Guidelines for determining the severity of vulnerabilities.
    - **On-Chain Assets**: A registry of relevant on-chain assets.
    - **Service Level Agreement (SLA)**: Terms and agreements for audit quotas.
    - **Incidents Log**: A historical record of EVM contract vulnerability incidents, categorized by year.
